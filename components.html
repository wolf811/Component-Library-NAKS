Оповещения с функцией "закрытия"

Используя JavaScript, можно закрыть оповещение. Для этого необходимо:

  • Убедится, что вы загрузили необходимый Bootstrap плагин (alert plugin).
  • Добавить кнопку закрытия с классами .alert-disissible и .close, которые размещают кнопку справа от текста оповещения.
  • На кнопке «Закрыть» добавить атрибут data-reject = «alert», который запускает функции JavaScript. С атрибутом обязательно используйте элемент <button> для правильного поведения на всех устройствах.
  • Для анимации оповещения при закрытии обязательно используйте классы .fade и .show.

Buttons (Кнопки)

Использование классов-модификаторов

Классы .btn могут быть использованы не только с элементом <button>, но и с элементами <a> и <input>.

Link

Добавив классы .btn-lg или .btn-sm можно изменить размер. Добавив .btn-block кнопка займет всю ширину родительского блока.

Для объединения кнопок в группу необходимо обернуть несколько .btn в .btn-group, для вертикального расположения вместо .btn-group используется .btn-group-vertical

Для изменения размера группы кнопок необходимо к .btn-group добавить соответствующий класс .btn-group-lg или .btn-group-sm.
