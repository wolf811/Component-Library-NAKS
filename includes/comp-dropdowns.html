Расположения выпадающего меню

Для изменения расположения выпадающего меню относительно кнопки необходимо добавлять следующие классы или атрибуты:

  • добавив к классу .dropdown-menu класс .dropdown-menu-right - меню выровняется по правому краю;
  • добавив к кнопке атрибут data-offset="10,20", можно сместить выпадающее меню согласно значений, заданных атрибутом;
  • к классу .btn-group добавить класс .dropup - меню сверху;
  • к классу .btn-group добавить класс .dropleft - меню слева;
  • к классу .btn-group добавить класс .dropright - меню справа.

Некоторые дополнительные возможности выпадающего меню

  • Форма вместо привычного меню
  • Активный пункт меню с классом .active
  • Неактивный пункт меню с классом .disabled
  • Возможность добавить заголовок с классом .dropdown-header или текст с классом .dropdown-item-text
  • Возможность добавить разделитель между пунктами меню с помощью div с классом .dropdown-divider
