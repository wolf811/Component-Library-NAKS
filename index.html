Обзор

Большинство компонентов и опций для создания проектов заимствовано из фреймворка Bootstrap 4, включая контейнеры для упаковки элементов, сетку (grid system), медиа запросы (@media...), утилиты адаптивности и прочие полезности.

Контейнеры

Контейнеры являются основным элементом компоновки и требуются при использовании сетки по умолчанию. Можно выбрать из адаптивного контейнера с фиксированной шириной (max-width) и контейнера шириной в 100% всего окна браузера.

Контейнеры могут быть вложенными, но в большинстве случаев этого не требуется.

Контейнер с фиксированной шириной

class = "container"

Контейнер с шириной в 100%

class = "container-fluid"

Сетка (Grid system)

Система сетки построена с помощью Flexbox, состоит из двенадцати колонок, использует серию контейнеров, строк и столбцов для компоновки и выравнивания содержимого. Ниже приведен пример объеденения сетки.

В сетку входят пять уровней предопределенных классов для построения сложных адаптивных макетов. Настройте размер ваших колонок на дополнительных маленьких, малых, средних, больших или дополнительных больших устройствах, как вы считаете нужным.
